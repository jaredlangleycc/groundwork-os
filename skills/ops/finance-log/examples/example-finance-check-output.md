## What this output models

### Specificity, not summary

Every line item has a vendor name and a dollar value. The user does not have to remember what "miscellaneous software" means. The CSV is the source; the report is the cross-section.

### Direction over magnitude

"Direction: flat after April bump" matters more than "Revenue: $2,000." The user already knows the magnitude. The skill names the trend.

### The early-operator concentration note

Three clients with 80% concentration is the most important sentence in the report for an Early Operator. Surface it as its own block, not buried in honest-mirror.

### The honest-mirror is not a verdict

The skill states "$143/mo of headroom against flat revenue" and asks a question. It does not tell the user to spend the headroom. It does not tell the user not to. The user owns the call.

### One closing question

The report ends with one question. The question is specific (timeframe, comparison anchor). The user can answer it. The skill does not stack three follow-ups.

## Alternate output: zero income month

If May had been a zero-income month (e.g., between client engagements):

```

  Zero income this month. Burn $187. Trend over 3 months: $1,600 → $2,000 → $0.
```

The skill does not catastrophize. It states the math. The question is forensic, not forward.
