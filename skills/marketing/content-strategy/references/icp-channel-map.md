# ICP Channel Map

Where common solopreneur ICPs actually consume content. Use to anchor Step 1 of the strategy build. Override with what the user knows from their own client interviews. This is a starting map, not a verdict.

## B2B SaaS roles

### Frontline Sales Manager (under 18 months in role)

- Primary: LinkedIn personal feed
- Secondary: Targeted Slack communities (Pavilion, Revenue Collective, RepVue Discord)
- Tertiary: Substack and Beehiiv newsletters (Cooper Press, Pavilion, Sales Hacker)
- Almost never: Threads, TikTok, IG
- Time-of-day patterns: 7-9 am scroll, 12-1 pm lunch scroll, 9-10 pm wind-down on LinkedIn

Anchor channel: LinkedIn personal.

### B2B SaaS founder ($1M to $10M ARR)

- Primary: X (formerly Twitter), LinkedIn personal
- Secondary: Substack newsletters, podcast subscriptions
- Tertiary: YouTube (deep tutorials)
- Almost never: TikTok, IG
- Time-of-day patterns: early morning X scroll, evening LinkedIn

Anchor channel: X + LinkedIn personal (one of two).

### VP of Sales (Series A to Series D)

- Primary: LinkedIn personal, executive newsletters
- Secondary: Industry conferences and recordings
- Tertiary: Podcasts on commute
- Almost never: TikTok, Threads
- Time-of-day: early morning, between meetings

Anchor channel: LinkedIn personal.

## How to use this map

1. Identify the closest ICP profile above.
2. Use the "Anchor channel" line as the user's primary.
3. Pick one secondary channel for repurpose.
4. Cut everything else from the strategy for now.
5. If the user's ICP doesn't appear above, ask the user to describe the ICP's day, then place them by inference.

## When this map is wrong

The map is wrong when:

- The user has already validated a different channel works (DMs, signups, calls trace back to channel X). Trust the data over the map.
- The user has a specific niche where their ICP gathers (e.g. a Discord, a Slack, a private forum). Always pick the smaller, more concentrated audience over the bigger general one.
- The ICP is geographic (real estate, local services). Local Facebook groups and Nextdoor beat global IG every time.
